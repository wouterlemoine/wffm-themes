Theme Creator for WFFM 2.3

  • What is this?

    This is a free tool that creates a theme for the WFFM-module in Sitecore.
    You can find it on Github: github.com/wouterlemoine/wffm-themes.

    It has the following principles:

    • Resets all default wffm-styling (so no need to remove it).
    • Responsive and mobile-friendly.
    • More custom styling can be added on top of this.
    • It covers all native formelements of WFFM 2.3.
  • Why?

    Though WFFM provides many layout options and css-theme-possibilities, the default-styling was ugly and not mobile friendly. It really needs additional css.

    This Theme-Creator has grown out of an css-library.

    Examples of themes: Delhaize Iglo Other

  • How to use?
    1. Go to the theme creator and create your theme.
    2. When done, download the CSS and put in the bottom of the CSS-file of your website. Or use the theme-files that WFFM provides: more info about the themes in the module (pdf - page 75)
    3. You only need the javascript file if you:
      • use the 'movinglabels' (advanced settings)
      • if you want the upload-button to display the selected files
